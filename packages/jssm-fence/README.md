# jssm-fence

Markdown fence rendering for [jssm](https://github.com/StoneCypher/jssm) state
machines — renders `fsl`-tagged code fences in Markdown into diagrams.

```
npm install jssm-fence
```

This package split out of the `jssm` package (previously available as the
`jssm/fence` subpath) in v6.

For documentation, issues, and the full monorepo, see
[github.com/StoneCypher/jssm](https://github.com/StoneCypher/jssm).
