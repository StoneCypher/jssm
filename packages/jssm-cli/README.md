# jssm-cli

Command-line tools for [jssm](https://github.com/StoneCypher/jssm) and the
FSL state-machine language — provides the `fsl`, `jssm`, `fsl-render`, and
`fsl-export-system-prompt` binaries.

```
npm install jssm-cli
```

This package split out of the `jssm` package's CLI bins in v6.

For documentation, issues, and the full monorepo, see
[github.com/StoneCypher/jssm](https://github.com/StoneCypher/jssm).
