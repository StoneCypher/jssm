Documentation

The composition
A real <fsl-instance> hosting <fsl-viz> and <fsl-editor> in its slots. The instance owns the layout and gutter; editing writes back, the machine rebuilds, and the graph follows. No glue code.
Editing & autocomplete
Ctrl+Space after a : for context completions — keys at the start of a line, values (shapes, colours with swatches) after a key.
Layouts
The View menu chooses how the panes sit: side by side or stacked (editor on either side), a single pane, tabbed, or Auto by window shape. Drag the gutter to resize; double-click to reset.
Highlighting
Beyond syntax colours, the editor reads the parsed machine: colour values get a swatch chip, state names are tinted, and shape / enum values are marked.
Theme
The ☀ / ☾ buttons switch light / dark; the editor re-themes live via the --fsl-* tokens.
