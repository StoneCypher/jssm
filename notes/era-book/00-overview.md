# The Program in One Page — 5.x → v16

FSL/jssm is one machine language growing through eleven named majors, each a complete,
announceable machine. The names read as a sentence: the machine computes, takes shape,
becomes transactional, portable, composable, proven, durable, trusted, ubiquitous, public.

**5.x The Long Goodbye (~25):** the 5-line ends. Triage fixes, dragon-suite prep, audit
re-verification, all as ordinary patches while the 6.0 assembly happens beside them.

**v6 The Ground (~40):** one release kills the dual-track. The breakage batch (atom charset,
`jssm-*` synonyms, probabilistic split, bare-functions API with `jssm/compat`), the One Merge
bringing the v6 branch home (vals, tapes, replayer, codegen) as the payload, the dragon lane
live, and the two paper keystones pinned: the canonical source hash and the operational
semantics. Trunk-based forever after.

**v7 The Computing Machine (~45):** the scalar expression tier — numeric tower, operators,
`if`/`let`/`case`, strings — and `where` guards, which cover most real-world gating with
scalars alone. Opens with the peggy-fork/generator-dual workshop renovation.

**v8 The Structured Machine (~70):** containers, ADTs, streams (fsl#1400 is the demo),
function-typed slots, and the §19 group/cascade system. Dragons-egg must be fully closed by
v6's end; from here every grammar slice lands dragon-covered at birth.

**v9 The Transactional Machine (~85):** the deep one. `assign` + undo journal + rollback, the
contract quartet, the error model, RTC formalized per C2, the hook registry — opening with
the mDT→val-record breakage batch. Safety checking and certificates (M1) begin here.

**v10 The Portable Machine (~70):** the contract cashed — conformance corpus at scale, IR
freeze, the Rust second implementation replaying every accumulated vector byte-identically,
byte-level reversibility and `format`, typegen. "One machine, two runtimes, provably同."

**v11 The Composable Machine (~80):** channels, systems, factories + seed tree, populations,
routes, supervision. The RCT-park flagship — and the manual's spine — become tellable.
Breakage: alias removal, strict mode.

**v12 The Proven Machine (~38):** temporal properties via Dwyer patterns, checker backends
(nuXmv/Spin/Storm), composed checking, the testing toolkit, behavioral diff, signed
certificates. The ladder decision record gates the upper rungs.

**v13 The Durable Machine (~17):** Temporal-class durable execution, verified — persistence
contract, recorded effects, `migrates from`, drift, `calibrate`. PROCESS-gated spec first.

**v14 The Trusted Machine (~11):** the trust economy — keyless signing, transparency log
(equivocation-hardened), receipts, lockfiles. Security-adversarial review gates the spec.

**v15 The Ubiquitous Machine (~28):** the host fleet against the v6/v13 contract, T2/T3
certification, adapters with golden traces, the WASM product. The `fsl` org era.

**v16 The Public Machine (~135, largely ambient):** registry, guardrails-as-product, `query`,
the curriculum/eval corpus, and the distribution program. Other people's machines — which is
what every earlier era existed to make safe.

Counts are topic unions (±10%). Governance: `notes/superpowers/eras/` — HANDOFF, INDEX,
briefs, ledgers, milestones fsl#48–#59. Everything decided is written; everything written
replays.
