# v16 — The Public Machine (one page)

**Mission:** other people's machines. Every earlier era existed to make a stranger's machine
safe to trust; this one builds the places where strangers meet.

**Contents (~135, largely version-agnostic/ambient — much can run alongside any era):** the
**registry**, loop-first per registry-close option B: prove the vertical
(agent states contract → queries → selects under policy → assembles → green proof or minimal
counterexample → swaps under refinement) on a tiny corpus BEFORE the metadata economy;
then trust tiers (target-qualified, multi-dimensional — never one "verified" badge),
third-party adjuncts, behavioral semver; **guardrails-as-product** (`mcp --policy guard.fsl`
— tool surfaces gated by certified machines; "proven, not prompted"); **`query`** fleet
governance with witness tapes; **semantic patches** (the multi-agent collaboration door);
`infer`/`synth` (aspirational bookends: learn from behavior / derive from properties); the
**curriculum + eval corpus at scale** (learn-by-repairing as the on-ramp AND the eval set);
the certified stdlib as public catalog; the **distribution program** — the migrated
jssm#831–#869 block: MCP registries, skills/plugins, llms.txt + cheat-sheet, machines.txt
(#778) + package exposure (#610), Linguist, articles/SO/Reddit/Wikipedia, apps; the manual/
book shipped whole; the Playwright announcer (fsl#1252, lol) posting releases.

**Exit:** an agent finds a proven debounce instead of writing one; a stranger's PR arrives
with a certificate; the corpus teaches new models FSL without us in the room.

**Hazards:** overclaiming is the ecosystem-killer — every trust word stays target-qualified
and evidence-backed (registry-close's "claims to avoid" list is law here); distribution is a
flywheel, not a launch — start it whenever, never let it gate engineering.

**Milestone:** fsl #59. **Sources:** eras-2-to-7 brief (era 7); W12.1–.10; megaspec §25.1;
registry-close.
